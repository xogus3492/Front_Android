# Intro
FYB(Fit Your Balance) 앱은 모든 사용자의 앱 내 활동을 수집한 데이터를 기반으로 각 사용자에게 맞춤 쇼핑몰을 추천해 주는 서비스를 제공합니다.

또한, 사용자의 신체 정보를 기반으로 3D모델을 구현하여 직접 옷을 입혀볼 수 있는 서비스를 제공합니다.
